Input Group Collection

InputGroupCollection is a simple javascript plugin to create partial input groups from templates. The generated inputs can be sent as input array or a single input with a json array as value.
InputGroupCollection in Github.

Examples
  • Basic input collection
  • Mix Two templates in one input collection
Elements
  • A wrapper to include container and button/s.
    • Container: Store generated collection input groups.
    • Buttons: create new collection input groups.
  • A template for the input group to clone
  • The navbar (Inside template or as standalone template) to show text and icons to
    • Delete collection.
    • Reorder collections
    • Show/hide input group

The template
The class .fic-group contains the input data
The input tag has a required data-name attribute to define the later input name attribute, so never include the name attribute on input tag.
For a single template, include navbar in template code.

How it works

To create a instance: 

    new InputGroupCollection('.wrapperName', {optional parameters...});
The optional parameters could be added as dataset.

With javascript

        new InputGroupCollection('.wrapper-selector', {
            template:'myTemplate',
            prefix:'myPrefix',
            container:'myContainer',
            jsonToNavbar:(json, wrapper)=>{
                //return string
            }
        });

Add a jsonToNavbar callback to generate a custom navbar text.
Parameters always overwrite dataset values.
