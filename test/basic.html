Basic input collection

  • Each data Wrapper contains dataset tags to set the configuration.
  • The form contains 3 form input collections with 3 modes using the same template.
    • "prefix" mode generate fields like "prefix[index][inputName]=value".
    • "json" & "jform" mode generate a hidden field to send one input as json.
  • The difference between "json" and "jform" is the hidden input name.
    • "jform" append the form name to prefix "myFormName[myPrefix]=[{collection1},...]".
    • "json" use only prefix "myPrefix=[{collection1},...]".
  • The generated input groups in "json" and "jform" mode do not have a name property, so they are not send in requests.
